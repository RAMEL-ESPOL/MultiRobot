Software package and ROS2 wrappers of the \href{http://www.sciencedirect.com/science/article/pii/S0031320314000235}{\texttt{ Aruco}} Augmented Reality marker detector library.\hypertarget{md_README_autotoc_md2}{}\doxysubsection{Features}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item High-\/framerate tracking of AR markers
\item Generate AR markers with given size and optimized for minimal perceptive ambiguity (when there are more markers to track)
\item Enhanced precision tracking by using boards of markers
\item ROS2 wrappers
\end{DoxyItemize}\hypertarget{md_README_autotoc_md3}{}\doxysubsection{Applications}\label{md_README_autotoc_md3}

\begin{DoxyItemize}
\item Object pose estimation
\item Visual servoing\+: track object and hand at the same time
